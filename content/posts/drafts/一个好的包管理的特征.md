## 好的模板
包括好的目录结构，以及好的格式化、项目设置。

包括：

- CMakeLists
- .clangd
- clang-format
- justfile
- src
- README
- toolchain
- .git

## 好的配置文件

配置文件应当简洁合理，采用主流的配置格式，如toml、yaml。对于功能：

```toml
[dependencies]
glfw = {git = "https://github.com/xxx/xxx.git"}
glm = {git = "https://github.com/xxx/xxx.git", root = "./third_party/glm/"} #自定义root，没有则自动创建
```
